# Mechanistic Air Emissions Simulator - MAES

## Quickstart

### Requirements
* Python 3.11 or later
* Git
* Python environment manager (e.g., Anaconda, Miniconda, pipenv or virtualenv)

### Installation
To install MAES, clone the repository and install the required packages using the following commands:
```bash
git clone https://github.com/CSU-METEC/MAES && cd MAES
```

```bash
pip install -r requirements.txt
```

### Usage
To run MAES, navigate to the repository and run the following command:
```bash
python MEETMain.py -s MEET2/ConstantSeparator.xlsx
```
