MEET Output File -- fluidFlows.csv
==================================

The fluidFlows.csv file identifies specific Fluid Flows used in EMISSION events.  Not heavily used.

Fields
------

ffSerialNumber
  Index from event log.
  
ffName
  Type of fluid.
  
ffType
  Implementation type of fluid flow.  Used for debugging.
  
ffGC
  Pointer to Gas Composition.
  
ffGCUnits
  Units of Gas Composition.
  
ffSecondaryID
  Secondary ID for Fluid Flow.
