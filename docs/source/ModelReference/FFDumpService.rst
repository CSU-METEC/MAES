
Fluid Flow Dump
===============

Model Filename: FFDumpService.json

Dump all Fluid Flow rates at a specific time, useful for debugging.  For a timeseries of specific fluid flows, use a Probe.

Site Definition Columns
-----------------------

**DumpID**
  Name of the output file of the specific dump

**Dump Time**
  Timestamp of all fluid flow dumps of all equipment

  *Units:* s
