
Fluid Flow links
================

Model Filename: FFLink.json

Define links between major equipment.

Site Definition Columns
-----------------------

**Outlet FacilityID**
  Facility ID of the upstream equipment

**Outlet UnitID**
  Unit ID of the upstream equipment

**Inlet FacilityID**
  Facility ID of the downstream equipment

**Inlet UnitID**
  Unit ID of the downstream equipment

**Flow Name**
  Condensate, Water or Vapor flow

**Flow SecondaryID**
  Special IDs to define specific flows
